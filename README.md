# daniu_weather

这是一款使用Flutter完成的天气APP，它使用了由[https://www.sojson.com](https://www.sojson.com)提供的免费接口。

请注意，该应用仅供参考，因为天气信息存在较大的延迟，不建议作为日常使用的天气预报工具。
